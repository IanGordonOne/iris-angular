About Iris

Iris is a social networking application that stores everything on its users' devices which communicate directly with each other — no corporate gatekeepers needed.

Longer description on GitHub.

Available at

  • iris.to
  • iris.cx
  • irislib.github.io
  • Browser extension: use Iris even if you are offline. In the future, can be used to sign in to websites.
    • Chrome
    • Firefox
  • Upcoming: Electrum desktop app with bluetooth and LAN peerfinding!

Privacy

It's important to remember that as for now, all posts, ratings and verifications are public.

Private chats can't be read by others, but anyone can see who are communicating with each other. "Last online" status is publicly available.

See privacy policy for more information.
