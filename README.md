## Development
### Requirements
Node >= 10, yarn (or npm)

### Get started
```
git clone https://github.com/identifi/identifi-angular.git
cd identifi-angular
```

### Install dependencies & build
```
yarn
```
Build goes into dist directory and browser extension into dist.zip.

### Just build
```
yarn build
```

### Develop
Run development server with browsersync:
```
yarn run serve
```
